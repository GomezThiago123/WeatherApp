COMO CORRER LA WEATHER APP

Requisitos:
- Node.js LTS
- npm
- Expo Go compatible con SDK 55

Instalar dependencias:
npm install

Iniciar el proyecto:
npm start

Probar en Expo Go:
npx expo start -c --lan

Si el celular no conecta por Wi-Fi:
npx expo start -c --tunnel

Probar en navegador:
npm run web

Verificar el proyecto:
npx tsc --noEmit
npm run lint
npm run tests

Notas:
- La app usa datos reales de Open-Meteo.
- No necesita API key.
- No subir node_modules al repositorio.
